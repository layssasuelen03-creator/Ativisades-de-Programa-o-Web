# CursoFeedback

Sistema simples de coleta de feedbacks de alunos para um curso online, desenvolvido com **Node.js** e **Express.js**. Os dados são mantidos em memória durante a execução do servidor.

---

## Como rodar o projeto

1. Clone o repositório ou baixe os arquivos
2. Acesse a pasta do projeto pelo terminal:
   ```bash
   cd "Atividade 5 - Node.js e Express"
   ```
3. Instale as dependências:
   ```bash
   npm install
   ```
4. Inicie o servidor:
   ```bash
   node server.js
   ```
5. Acesse no navegador:
   ```
   http://localhost:3000/
   ```
